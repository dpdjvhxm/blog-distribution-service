# Blog Distribution Service

블로그 배포 서비스를 운영하기 위한 내부 관리용 SPA입니다.

## 포함된 화면

- 로그인
- 메인 대시보드
- 배포 접수
- 배포 관리
- 계정 관리

## 주요 기능

### 로그인

- 관리자 로그인 화면
- 데모 계정 표시
- 로그인 이후 관리 화면 진입

### 대시보드

- 전체 접수 건수
- 진행중/완료 건수
- 전체 배포 진행률
- 플랫폼별 접수/배포량 차트
- 상태별 접수 현황 차트
- 최근 배포 요청
- 계정 상태 요약

### 배포 접수

- 캠페인명 등록
- 메인 키워드 등록
- 플랫폼 선택
- 배포 수량 입력
- 담당자 입력
- 마감일 입력
- 우선순위 선택
- 운영 메모 입력

### 배포 관리

- 접수 목록 조회
- 캠페인/키워드/플랫폼/담당자 검색
- 상태 변경
- 발행 수량 +1 처리
- 진행률 자동 표시
- 완료 처리 시 배포 수량 자동 반영

### 계정 관리

- 블로그/커뮤니티 계정 등록
- 플랫폼/담당자/상태 관리
- 누적 발행 수 및 최근 사용일 표시
- 계정 상태 변경

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 로컬 주소로 접속하면 됩니다.

## 빌드

```bash
npm run build
```

## 현재 구현 상태

현재 버전은 프론트엔드 데모 버전입니다. 데이터는 React 상태로만 관리되며, 새로고침 시 초기 샘플 데이터로 돌아갑니다.

향후 확장 시 다음 기능을 추가하면 됩니다.

- Supabase 또는 Firebase 기반 로그인/DB 연동
- 실제 권한별 메뉴 제어
- 배포 URL 등록 및 검수 이력 관리
- 원고 파일/이미지 첨부
- 계정 중복 체크
- 배포 실패/누락 AS 관리
- CSV/엑셀 다운로드
- 월별/담당자별 성과 리포트
